INSTALLATION STEPS
------------------
1. Create a folder in your WAMP www directory:
   C:\wamp64\www\portal-method-enhanced\

2. Save all provided files in this directory:
   - index.php (Enhanced landing page)
   - regular.php (Enhanced regular frame solver) 
   - irregular.php (Enhanced irregular frame page)
   - theory.php (Educational content)
   - manifest.json (PWA configuration)

3. Start WAMP server and ensure:
   - Apache is running (green icon)
   - PHP is enabled
   - MySQL is running (for future database features)

4. Access the application:
   http://localhost/portal-method-enhanced/

5. For PWA features (optional):
   - Add web app icons (icon-192.png, icon-512.png)
   - Enable HTTPS for full PWA functionality

USAGE INSTRUCTIONS
------------------
1. First Time Use:
   - The tutorial will automatically start
   - Follow the guided steps to learn the tool
   - Create your first project

2. Creating Analysis:
   - Select "Regular Frame Solver"
   - Enter frame geometry (1-5 storeys, 1-5 spans)
   - Input storey heights and loads
   - Choose load case type
   - Click "Solve" for analysis

3. Project Management:
   - Name your project in the header
   - Use "Save Project" to store analysis
   - Load previous projects from home page
   - Export professional reports

4. Code Compliance:
   - Review automatic compliance checks
   - Address any warnings or errors
   - Select appropriate design code

5. Exporting Results:
   - Use "Professional Report" for PDF
   - "Export Excel" for spreadsheet data
   - All exports include compliance summary

TROUBLESHOOTING
---------------
1. If pages don't load:
   - Check WAMP server status
   - Verify file locations
   - Check browser console for errors

2. If validation doesn't work:
   - Ensure JavaScript is enabled
   - Check browser compatibility

3. If sessions don't persist:
   - Check PHP session configuration
   - Verify write permissions
